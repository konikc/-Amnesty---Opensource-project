## Быстрый старт

### 1. Готовый EXE (после сборки)

```
dist\SHelper.exe  →  ПКМ → Запуск от имени администратора
```

### 2. Сборка из исходников

Требования: **Python 3.10+** (с галочкой "Add Python to PATH")

```batch
build.bat
```

После выполнения батника готовый `SHelper.exe` появится в папке `dist\`.

### 3. Запуск из исходников напрямую

```batch
python shelper.py
```

## Запуск в WinRE

Если система полностью заблокирована:

1. Загрузитесь с установочной флешки Windows
2. `Восстановление системы` → `Командная строка`
3. Перейдите к файлу `SHelper.exe` на USB
4. `SHelper.exe` → ПКМ → Запуск от имени администратора

## Сборка EXE

Батник `build.bat` выполняет автоматически:

1. Проверку Python
2. Установку PyInstaller
3. Генерацию иконки (`create_icon.py`)
4. Сборку `dist\SHelper.exe` с флагами:
   - `--onefile` — один файл
   - `--windowed` — без консоли
   - `--uac-admin` — запрос прав при запуске
   - `--icon` — иконка проекта

## Безопасность

> ⚠️ SHelper — мощный инструмент. Используйте ответственно.

- Операции с реестром применяются немедленно
- Восстановление MBR/BCD затрагивает физический диск
- Замена sethc/utilman открывает cmd на экране блокировки
- Рекомендуется создать точку восстановления перед работой
